# vSOC Alert Specification

## 1. Alert Overview
This specification defines the JSON schema and taxonomic mapping for alerts generated by the Multi-Layer Automotive IDS.

## 2. JSON Alert Schema
Each detection generates a standardized alert for ingestion by a Vehicle Security Operations Center (vSOC).

```json
{
  "timestamp": "ISO-8601 string",
  "vehicle_id": "Unique VIN or Identifier",
  "severity": "Critical | Suspicious | Informational",
  "confidence": "1 (Single Layer) | 2 (Multi-Layer)",
  "source": "Multi-Layer IDS (CAN + Telematics)",
  "mitre_mapping": "MITRE ATT&CK for Automotive ID",
  "details": {
    "can_anomaly": "Boolean",
    "behavioral_deviation": "Boolean",
    "attack_type": "DoS | Fuzzy | Spoofing | Behavioral"
  }
}
```

## 3. Severity Tiers
| Severity | Definition | Multi-Layer Context |
| :--- | :--- | :--- |
| **Critical** | Immediate safety threat. | Overlapping CAN anomaly AND behavioral deviation. |
| **Suspicious** | High probability of attack. | Strong CAN-level detection (e.g., DoS sequence). |
| **Informational** | Low-level anomaly. | Single-source timing jitter or outlier. |

## 4. MITRE ATT&CK Mapping
Our IDS maps all detections to the specialized MITRE ATT&CK for Industrial Control Systems / Automotive taxonomy.

| Attack Type | MITRE ID | Technique Name |
| :--- | :--- | :--- |
| **DoS** | T0855 | Unauthorized Command Message |
| **Fuzzy** | T0830 | Adversary-in-the-Middle |
| **Spoofing** | T0843 | Replay Attack |
| **Behavioral**| T0828 | Loss of Control |

## 5. Analyst Response Workflow
1. **Critical Alert:**
    - Immediately isolate the vehicle's telematics connection.
    - Initiate Safe State (e.g., limp mode via OTA command).
    - Review telematics-can data correlation logs for forensics.
2. **Suspicious Alert:**
    - Increase monitoring resolution on the specific VIN.
    - Correlate with other vehicles in the same fleet for fleet-wide attack detection.
3. **Informational Alert:**
    - Store for longitudinal behavioral analysis.
    - No immediate action required.
